# SHelltide - Version: 1.0.0B

## Description
SHelltide was designed for full automation of Helltide, currently in BETA.
- Explore the helltide
- Fights Elites/Champions/Boss enemies
- Collect Chests
- Collect Silent Chests
- Backtracking for Missed Chests
- Performs the Madein ritual
- Supports Alfred
- and more...

## Requirements
You must have the following scripts for the script to work properly
- Alfred: The Butler by Leoc V1.3.5
- Looter by Letrico V1.2.6


# Configurations
### Menu SHelltide
- Enable --> Activates the SHelltide script
- Enable Debug --> Activates for debug view

### SubMenu Info / Status - Settings
- Info offset X --> Change the position of the Info Menu on the X-axis
- Info offset Y --> Change the position of the Info Menu on the Y-axis

### SubMenu Alfred - Settings
- Salvage with Alfred --> Allows Alfred to be triggered

### SubMenu Maiden - Settings
- Enable --> Allows the script to perform the Maiden ritual
- Return to the point before the trigger --> Return to the “original” position after performing the ritual (Raccomamended keep it on)
- Attempts Slider --> Number of executions to trigger Maiden
- Trigger Time --> After how long it triggers to perform the Maiden ritual

### SubMenu Chests - Settings
- Open Silent Chests --> Allows the script to collect silent chests (if you have the key)
- Return to the point before the trigger --> Return to the “original” position after performing backtracking for missed chests (Raccomamended keep it on)
